# PUMS input files
These files are needed from the Census Bureau but are not included in the  repository for space considerations:
- ss09pwa.csv
- ss09por.csv
- ss09pnv.csv
- ss09pid.csv
- ss09pca.csv
- ss09hwa.csv
- ss09hor.csv
- ss09hnv.csv
- ss09hid.csv
- ss09hca.csv

Users can download all of the necessary files by running `download.bash` in a bash shell.
